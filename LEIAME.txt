O exercicio programa foi desenvolvido no IDE CodeBlocks, e por conta disso o projeto no formato especifico para o CodeBlocks foi enviado junto com o codigo-fonte. 
Por alguma razao, o CodeBlocks, por vezes, nao inclui todas as classes no projeto, entao se for rodar o codigo nessa IDE ter certeza de que todos os arquivos estao no projeto.

Este exercicio programa foi desenvolvido com base numa classe Matrix que centraliza algumas
operacoes realizadas com matrizes. Essa classe e' utilizada como classe pai para as classes
que inicializam as matrizes relativas 'a primeira tarefa, as quais tem um algoritmo especifico
para preenchimento de seus valores. Alem disso, ha' as classes NullMatrix, que gera uma matriz
nula, RandomPositiveMatrix, que gera uma matriz aleatoria com valores positivos entre 0 e 10, e
UserGeneratedMatrix, que inicializa uma matriz com valores digitados pelo usuario, todas subclasses
de Matrix.

Para executar algum exercicio, a funcao main presente no arquivo main.cpp deve ser modificada, de modo a DESCOMENTAR a funcao que realiza tal exercicio.
No segundo exercicio, os valores desejados devem ser inseridos pelo usuario.
Para mudar os hiperparametros do terceiro exercicio, e' necessario mudar os valores definidos no inicio do main.cpp pelos #define realizados.

O programa deve ser compilado com g++ em concordancia com a versao C++11 (-std=c++11) e com a flag de otimizacao -O2, para acelerar o processo.
